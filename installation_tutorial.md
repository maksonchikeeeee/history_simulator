# Туториал по установке
## 1. Создайте отдельную папку
papka
## 2. Добавьте в эту папку файл main.py и создайте папку data (!важно, именно data)
papka
   main.py
   data
## 3. Добавьте в папку data все файлы из такой же папки на гитхабе
papka
   main.py
   data
     dates_answers_ru.txt
     dates_questions_ru.txt
## 4. Все теперь можете открывать папку в любой IDE и запускать main.py
