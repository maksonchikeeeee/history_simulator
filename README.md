# History Simulator
Приветствую, это тренажер для повторения знаний по Истории!  
Здесь будет возможность повторить западающие темы или просто узнать что-то новое.  
Приложение является консольным и очень сырым по содержанию


## Версия приложения v0.001 "Релиз":
- Сделано главное меню, система самой игры, режим "Россия" + "Даты".
- В режим "Россия" + "Даты" добавлено 25 вопросов во временном периоде от 1689 до 1754
